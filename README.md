# Sales Data Pipeline

## Overview
Built an end-to-end data pipeline using Databricks, Snowflake, and Power BI.

## Architecture
Raw Data → Clean → Transform → Snowflake → Power BI

## Steps
- Cleaned data using COALESCE
- Derived revenue and profit
- Applied GROUP BY for aggregation
- Loaded into Snowflake
- Built dashboard in Power BI

## Tools Used
- Databricks
- SQL
- Snowflake
- Power BI

## Output
- Revenue by Region
- Product Performance
- Profit Analysis
